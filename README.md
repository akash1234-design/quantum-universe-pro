# 🌌 Quantum Universe Pro — Data Analytics + Cosmology

An interactive data analytics portfolio project that blends **quantum physics simulations** with a full **data analytics dashboard**. Built for showcasing EDA, visualization, and storytelling skills for data analytics roles.

🔗 **Live Demo:** *https://quantum-universe-pro-kkgdt8cr5vauuelcexejqm.streamlit.app/*

---

## ✨ Features

### 📊 Data Analytics Dashboard
- Simulated galaxy survey dataset (3,000 galaxies)
- KPIs: Total galaxies, Avg redshift, Avg mass, Type distribution
- 3D interactive scatter plot
- Summary statistics table
- Correlation heatmap
- CSV / HTML export

### 🔭 Cosmology Simulations
- **Big Bang Timeline** — 0 to 13.8 Billion years slider with real-time expansion
- **3D Galaxy Universe** — Galaxies born from quantum fluctuations
- **Supermassive Black Hole** — Event horizon visualization
- **Quantum Fluctuations + NASA CMB** — Planck-like CMB map

### 📐 Physics + Analytics
Each module includes:
- Physics equations with LaTeX
- Plain English explanation
- Data insights

---

## 🛠️ Tech Stack

- **Python 3.10+**
- **Streamlit** — Web app framework
- **Plotly** — Interactive 3D visualizations
- **Pandas / NumPy** — Data analysis
